# Sync

## Setup

### System dependencies

`7z`, `git` and `python3` must be installed on your machine.

### Python dependencies

```bash
python3 -m venv venv
source ./venv/bin/activate
pip3 -r requirements.txt
```

## Init

```bash
python3 sync.py --init
```

## Sync

```bash
python3 sync.py --sync
```
